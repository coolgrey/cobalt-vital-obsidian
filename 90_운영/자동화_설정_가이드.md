# 자동화 설정 가이드

## 이번에 적용한 것
- Daily Notes 저장 폴더: `00_대시보드/일일로그`
- Daily Notes 템플릿: `_Template/오늘_실행로그_초미니`
- Core Templates 폴더: `_Template`
- Templater 템플릿 폴더: `_Template`
- Web Clipper import용 JSON 2종 추가

## 자동으로 되는 것
1. **오늘의 데일리 노트 생성**
   - Obsidian에서 `Open today's daily note`를 실행하면 오늘 날짜 파일이 자동 생성됨.
2. **초미니 템플릿 자동 삽입**
   - 새 데일리 노트가 만들어질 때 아래 2줄 구조가 자동 들어감.
     - `# 오늘 새 소스`
     - `# 다음 행동`
3. **웹 소스 메타데이터 자동 채움**
   - Web Clipper 템플릿을 import하면 제목, URL, 작성자, 발행일 같은 메타데이터를 자동 저장 가능.
4. **웹 클립을 오늘 데일리 노트에 바로 추가**
   - `건강계좌 소스 일일로그 추가` 템플릿을 쓰면 클립이 오늘 노트 하단에 바로 붙음.

## 자동으로 안 되는 것
- Obsidian 자체가 웹을 돌아다니며 **스스로 검색**해서 새 자료를 가져오지는 않음.
- 이 부분은 브라우저 Web Clipper 사용 또는 외부 자동화가 필요함.

## Web Clipper import 방법
1. 브라우저에 Obsidian Web Clipper 설치
2. Web Clipper 설정 열기
3. 템플릿 화면에서 **Import** 선택
4. 아래 파일 2개 import
   - `_Template/web_clipper/건강계좌_소스_신규노트.json`
   - `_Template/web_clipper/건강계좌_소스_일일로그추가.json`

## 추천 사용 흐름
### 흐름 A: 기사 하나를 소스 노트로 저장
- Web Clipper → `건강계좌 소스 신규노트`
- 저장 위치: `06_RAW/Web Clipper`
- 나중에 필요한 때만 위키 노트로 연결

### 흐름 B: 바쁜 날 10초 기록
- Web Clipper → `건강계좌 소스 일일로그 추가`
- 오늘 데일리 노트에 링크 한 줄만 추가
- 하루 끝에 `다음 행동` 한 줄만 적기

## 매일 해야 하는 최소 행동
- 오늘 데일리 노트 열기
- `오늘 새 소스` 또는 Web Clipper append 사용
- `다음 행동` 1줄 적기
