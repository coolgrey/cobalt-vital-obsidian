# 고립 노트 점검

## 목적

## 점검 항목
- [ ] 백링크가 0개인 핵심 문서가 있는가
- [ ] 프로젝트 문서가 공통 노트와 연결되어 있는가
- [ ] 제출용 문서가 원천/프로젝트 문서와 연결되어 있는가
- [ ] 새로 만든 OUTPUT 문서가 관련 노트에 연결되었는가
