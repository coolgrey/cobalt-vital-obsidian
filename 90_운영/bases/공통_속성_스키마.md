# 공통 속성 스키마

모든 핵심 노트에 아래 frontmatter를 가능한 한 통일해서 사용합니다.

```yaml
---
type: source | wiki | project | output | deadline
project: 건강계좌 | CocFlow | 공통 | 제출용
status: draft | active | done | archived
source_kind: web | paper | repo | dataset | meeting
source_url:
source_date:
updated:
confidence: high | medium | low
derived_from:
tags:
---
```

## 권장 사용 예시

### 원천 자료
```yaml
---
type: source
project: 공통
status: active
source_kind: web
source_url: https://example.com
source_date: 2026-04-09
updated: 2026-04-09
confidence: high
tags: [raw, web]
---
```

### 프로젝트 문서
```yaml
---
type: project
project: 건강계좌
status: active
updated: 2026-04-09
confidence: medium
derived_from:
  - [[01_공통/01_원천/시장분석]]
tags: [건강계좌, 전략]
---
```

### 제출용 문서
```yaml
---
type: deadline
project: 제출용
status: draft
updated: 2026-04-09
confidence: high
tags: [여성창업경진대회, 제출]
due: 2026-04-16 15:00
owner: 최지원
---
```
