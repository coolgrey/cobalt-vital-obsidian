# 07_OUTPUTS

이 폴더는 **질의 결과와 파생 산출물**을 저장합니다.

## 목적
- LLM 질의응답 결과를 위키에 다시 적재
- 발표 자료, 요약 보고서, 의사결정 메모를 누적
- 탐색 결과가 일회성 답변으로 끝나지 않도록 축적

## 하위 폴더 권장
- `qa_notes/` : 질문별 정리 문서
- `decision_memos/` : 의사결정 메모
- `slides/` : Marp 기반 발표 자료
- `figures/` : 차트, 도식, 이미지
- `reports/` : 주간/월간 요약 보고서

## 규칙
- 질문 결과는 가능한 한 Markdown 문서로 저장
- 중요한 결과는 관련 프로젝트 노트와 상호 링크
- 발표자료/도식도 결과물로 간주하고 다시 위키에 편입
