# 건강계좌사업 1페이지

## 1. 사업 한 줄 정의
건강검진·의료이력·라이프로그를 통합하여 개인이 자신의 건강 상태를 장기적으로 이해하고 관리할 수 있게 돕는 **건강계좌 기반 디지털 헬스 서비스**

## 2. 해결하려는 문제
- 현재 건강정보는 검진, 병원기록, 복약, 웨어러블, 생활습관 데이터로 흩어져 있음
- 사용자는 자신의 건강 상태를 한 흐름으로 이해하기 어려움
- 기존 서비스는 일회성 조회나 단편적 기록에 머무르는 경우가 많음
- 그 결과, 예방·관리·행동변화로 연결되지 못함

## 3. 타깃 고객
### 1차 타깃
- 건강검진 결과를 꾸준히 관리하고 싶은 성인
- 만성질환 전단계 또는 생활습관 관리 필요군
- 자신의 건강 데이터를 통합해서 보고 싶은 사용자

### 2차 타깃
- 기업 복지형 건강관리 도입 기업
- 공공 보건 실증 기관
- 향후 건강관리 파트너사

## 4. 핵심 해결책
- 건강정보를 한곳에서 모아 보는 **개인 건강계좌**
- 건강검진 결과, 의료이력, 웨어러블, 식단, 복약, 자가입력 데이터 통합
- 변화 추세를 보여주는 대시보드 제공
- 건강신용점수 또는 건강지표 기반 변화 리포트 제공
- 개인 맞춤형 행동 유도 및 관리 동기 부여

## 5. 차별점
- 단순 기록 앱이 아니라 **통합형 건강자산 관리 구조**
- 금융의 계좌 개념처럼 건강정보를 장기 흐름으로 관리
- 흩어진 데이터를 사용자 관점에서 이해 가능한 구조로 재구성
- 향후 공공·민간 데이터 연계를 고려한 확장 가능 구조

## 6. 왜 지금 필요한가
- 예방 중심 건강관리 수요 확대
- 디지털 헬스 및 개인 건강데이터 활용 관심 증가
- 웨어러블 및 자가 기록 기반 관리 습관 확산
- 공공/민간 건강정보 연계 환경이 점차 확대되는 추세

## 7. 시장/도입 경로
### 초기 진입
- 공공 실증
- 기업 복지형 파일럿
- 건강관리 관심 사용자 대상 MVP 검증

### 확장 경로
- B2G → B2B → B2C / B2B2C 확장
- 데이터 기반 리포트/구독/제휴 모델로 확장

## 8. 수익모델
- 기업/기관 대상 도입형 계약
- 건강관리 구독형 서비스
- 프리미엄 리포트 또는 맞춤형 분석 기능
- 향후 제휴형 데이터 서비스 확장 가능

- 통합된 정보가 행동변화 동기로 이어지는가
- 초기 타깃 고객이 실제 사용 의향을 보이는가

## 14. 30초 설명 버전
건강계좌사업은 건강검진, 의료이력, 웨어러블, 생활기록처럼 흩어진 건강정보를 하나의 계좌처럼 통합해 사용자가 자신의 건강 상태를 장기 흐름으로 이해하고 관리할 수 있게 돕는 디지털 헬스 서비스입니다. 단순 조회가 아니라 변화 추적, 건강지표, 행동 유도까지 연결하는 구조를 목표로 합니다.
