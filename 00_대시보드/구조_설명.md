# 건강계좌 볼트 구조 설명

이 볼트는 아래 3층 구조로 운영한다.

## 1. 소스
가공 전 재료를 모으는 층

대상 폴더
- `01_공통/01_원천`
- `05_리서치`
- `06_RAW`
- `Clippings`

예시
- 기사
- 정책 자료
- 경쟁사 메모
- 웹클리핑
- 인터뷰 메모
- 회의 원문

## 2. 위키
소스를 해석하고 연결하는 층

대상 폴더
- `01_공통`
- `02_건강계좌`
- `03_CocFlow`

예시
- 문제정의
- 고객정의
- 차별점
- 기능 구조
- 서비스 흐름
- 경쟁구도 해석

## 3. 콘트롤
바로 제출하거나 실행하는 층

대상 폴더
- `00_대시보드`
- `04_제출용`
- `07_OUTPUTS`
- `90_운영`

예시
- 실행보드
- 지원서 초안
- 발표용 요약
- 오늘의 데일리 로그
- 운영 체크리스트

## 기본 원칙
- 새 기사/자료는 먼저 소스로 넣는다.
- 의미를 해석한 문서는 위키에 둔다.
- 이번 주 바로 쓸 문서는 콘트롤에 둔다.

## 오늘 가장 먼저 볼 곳
- `00_대시보드`
- `00_대시보드/일일로그`
- `04_제출용`
