# 심평원 빅데이터/공공데이터 활용안 마련용 자료

공모전에서 특히 강조할 포인트
심사위원이 "공공데이터를 얼마나 실질적으로 활용하는가"를 보기 때문에, 단순히 "활용할 예정"이 아니라 아래처럼 구체화하면 훨씬 유리합니다.

HIRA 빅데이터 포털에서 샘플 데이터 실제 다운로드 후 항암 관련 상병코드 분포 분석 결과를 슬라이드에 수치로 제시
KCD 코드 → 자연어 증상 표현 매핑 테이블을 직접 구축했다는 것을 프로토타입 근거로 활용
"건보 데이터 기반으로 항암 부작용 상위 20개 증상을 추출해 RAG 문서 베이스를 구성했다"는 식의 서술

데이터 접근 방법 (현실적 순서)

HIRA 보건의료빅데이터 포털 ([opendata.hira.or.kr](http://opendata.hira.or.kr/)) → 공개 샘플 데이터는 회원가입 후 즉시 다운로드 가능
국립암센터 통계 → 암등록통계는 공개 자료, 즉시 활용 가능
식약처 KIDS → 의약품 부작용 보고 데이터 공개 API 존재
건보공단 표본코호트 → 연구 신청 필요, 시간이 걸리므로 공모전 이후 중기 계획으로 설정

공모전 단계에서는 1\~3번만으로도 충분히 "데이터 기반 시스템"임을 증명할 수 있습니다.
