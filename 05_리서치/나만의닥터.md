# 나만의닥터

## 분류
비대면진료 / 헬스케어 플랫폼

## 관찰 포인트
- 사용자가 병원/약/검진 정보를 얼마나 쉽게 확인하는가
- 건강정보 조회 이후 행동 유도 기능이 있는가
- 생활관리나 장기 건강관리로 연결되는가

## 건강계좌 관점 메모
- 진료 및 처방 정보 접근은 강점이 될 수 있음
- 그러나 건강자산 관점의 장기 축적 구조와 해석 경험은 별도 검토 필요
- 건강계좌는 "조회"보다 "장기 흐름 이해"를 더 강조해야 함
