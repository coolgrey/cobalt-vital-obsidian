# 킨스타워 입주신청 분석

**원본 자료**: [[06_RAW/0320_중장년_킨스타워]]

## 📋 개요

킨스타워 입주신청서 3개 버전(1차~3차)을 통해 사업계획서 작성 기준 및 개선 과정을 분석한 문서.

## 🎯 학습 포인트

### 1. 입주신청서 구성

입주신청서 전체 구성:
- 신청서 기본 정보
- 사업계획서 (핵심)
- 개인정보이용동의서

### 2. 사업계획서 개선 과정 (1차 → 3차)

| 버전 | 주요 개선 사항 | 심사 결과 |
|------|--------|---------|
| 1차 | 초기 아이디어 | 보완 요청 |
| 2차 | 시장 분석 강화, 수익 모델 개선 | 보완 요청 |
| 3차 | 전략 구체화, 실현 가능성 증대 | 통과 |

### 3. 심사에서 강조되는 요소

✅ **긍정 평가 요소**
- 명확한 시장 정의 및 타겟 고객
- 경쟁사 대비 차별점
- 구체적인 사업 단계별 계획
- 현실적인 수익 모델

❌ **보완 요청 요소**
- 추상적인 아이디어
- 시장 규모 미흡 또는 근거 부족
- 팀 역량 부족 설명

## 💡 건강계좌 적용 시사점

### 킨스타워 입주신청서에서 배울 점
1. **시장 정의 명확성**: "건강검진 이상소견자" 명확하게 세분화
2. **차별점 강조**: 단순 정보 조회 아님 → "자산화" 포인트 부각
3. **수익 모델 구체화**: B2C/B2B 혼합 모델 제시
4. **실현 가능성**: 현재 기술 및 규제 환경 고려

### 개선 권고사항
- [ ] 건강계좌 사업계획서: 킨스타워 3차 버전 수준으로 고도화
- [ ] 차별점 섹션: 경쟁사(카카오, 네이버) 대비 명시
- [ ] 수익 모델: B2B 파트너십 구체 예시 추가
