# 룰루메딕

## 분류
의료기록 / 건강정보 관리 서비스

## 관찰 포인트
- 검진/진료 정보의 통합 수준
- 사용자가 이해하기 쉬운 요약 제공 여부
- 건강관리 행동으로 이어지는 기능 존재 여부

## 건강계좌 관점 메모
- 건강정보 접근성과 편의성은 참고 가치가 큼
- 다만 건강계좌는 "건강정보를 자산처럼 축적하고 변화로 해석하는 구조"를 더 분명히 해야 함
- 단순 조회형 서비스와 차별화 포인트를 지속 정리 필요
