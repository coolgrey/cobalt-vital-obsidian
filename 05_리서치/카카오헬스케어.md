# 카카오헬스케어

## 분류
디지털 헬스케어 플랫폼

## 관찰 포인트
- 데이터 연결 범위
- 사용자 친화적 인터페이스
- 특정 질환/생활습관 관리와의 결합 방식
- 플랫폼 확장성

## 건강계좌 관점 메모
- 대형 플랫폼 기반 신뢰와 접근성은 강점
- 건강계좌는 개인 건강기록의 장기 흐름과 자산화 개념을 더 선명히 가져가야 함
- 검진/의료이력/생활기록을 하나의 "개인 건강계좌"로 묶는 설명력이 중요함
