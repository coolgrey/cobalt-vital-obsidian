# 의료AI 경진대회 분석

**원본 자료**: [[06_RAW/0514_심평원]]

## 📋 개요

2026년 보건의료빅데이터·AI 활용 창업경진대회 (심평원 주관) 참가 전략 분석

## 🎯 경진대회 개요

### 주최 기관
- **주관**: 건강보험심사평가원 (심평원)
- **목적**: 보건의료 빅데이터 기반 창업 아이디어 발굴 및 사업화 지원

### 참가 대상
- 의료 AI, 헬스테크 기업
- 초기 창업기업 및 예비 창업자
- 개인 및 팀 단위 모두 참가 가능

### 경진대회 규모
| 항목 | 규모 |
|------|------|
| 총상금 | 5,000만원 (예상) |
| 대상 상금 | 1,500~2,000만원 |
| 선발 팀 수 | 10~15팀 |
| 최종 진출팀 | 3~5팀 (본선) |

## 📊 평가 기준 분석

### ✅ 평가 항목

| 항목 | 배점 | 평가 포인트 | 건강계좌 관련성 |
|------|------|---------|---------|
| **기술 혁신성** | 30점 | AI/빅데이터 활용도, 알고리즘 차별성 | ⭐⭐⭐ 높음 |
| **의료 현장 적용성** | 25점 | 임상 유용성, 실제 진료 적용 가능성 | ⭐⭐⭐ 높음 |
| **사업화 가능성** | 25점 | 시장 수요, 수익모델, 규제 대응 | ⭐⭐ 중간 |
| **사회적 가치** | 20점 | 국민 건강 증진, 의료비 절감, 형평성 | ⭐⭐⭐ 높음 |

### 평가 프로세스
1. **1차 심사** (온라인): 사업계획서 기반
2. **2차 심사** (현장 피칭): 5분 프레젠테이션
3. **최종 심사** (투자자 피칭): 전문가 Q&A

## 💡 건강계좌 적용 전략

### 전략적 포지셔닝

#### 1️⃣ 기술 혁신성 (30점) — 강점 분야
**기술 차별화 요소:**
- **마이데이터 API 활용**: 개인 의료 데이터 통합 관리
- **AI 분석 알고리즘**: 건강검진 이상소견 예측 모델
- **개인화 추천**: 각 개인의 의료 이력 기반 맞춤형 조언

**제시 전략:**
- [ ] 기술 논문 또는 특허 출원 현황 제시
- [ ] 알고리즘 정확도 및 성능 메트릭 (AUC, Sensitivity 등)
- [ ] 비교 기술: 카카오헬스, 네이버 웰니스 대비 차별점

#### 2️⃣ 의료 현장 적용성 (25점) — 주력 분야
**임상적 유용성:**
- **타겟 환자**: 건강검진 수검자 1,700만 명 중 이상소견자
- **임상 시나리오**: 검진 후 추적 관리, 예방적 의료 개입
- **의료진 활용**: 의사와 환자 간 소통 도구 역할

**제시 전략:**
- [ ] 현재 베타 사용자 데이터 (N=?, 사용률, NPS 점수)
- [ ] 의료기관 협력 현황 (파트너 병원 목록)
- [ ] 예방의학 효과: 조기 발견 사례 또는 임상 결과

#### 3️⃣ 사업화 가능성 (25점) — 개선 필요
**시장 수요 입증:**
- **B2C 시장**: 개인 사용자 구독 모델
- **B2B2C 시장**: 기업 직원 건강검진 플랫폼 연동
- **B2B 시장**: 의료기관, 보험사, 건강검진센터 연계

**수익모델:**
- 개인 구독: 월 5,000원~10,000원
- 기업 복지: 월 1,000원~2,000원/명
- 데이터 기반 서비스: 건강보험사와의 협력

**제시 전략:**
- [ ] 구체적인 고객 확보 계획 (3개월 내 몇 명)
- [ ] 파일럿 프로젝트: 기업 고객 또는 의료기관과의 협력 사례
- [ ] 수익 모델의 현실성: 가격 전략 및 고객 생애가치(LTV)

#### 4️⃣ 사회적 가치 (20점) — 강점 분야
**국민 건강 증진:**
- **조기 발견**: 이상소견 조기 진단으로 질환 악화 예방
- **의료비 절감**: 예방적 개입으로 치료비 감소
- **의료 접근성**: 디지털 헬스케어로 의료 사각지대 해소

**형평성:**
- 저소득층도 접근 가능한 가격 정책
- 취약계층 대상 무료 제공 프로그램

**제시 전략:**
- [ ] ESG 관점의 사회 기여 명시
- [ ] 예상 영향 범위: 연 100만 명 이상 혜택
- [ ] SDG(지속가능개발목표) 연계: Goal 3 (건강과 웰빙)

## 📈 건강계좌 vs CocFlow 포지셔닝

| 항목 | 건강계좌 | CocFlow |
|------|---------|---------|
| **기술 초점** | 마이데이터 + AI 분석 | 진료 AI 생태계 |
| **대상 환자** | 이상소견 관리 | 전체 진료 효율화 |
| **심평원 경진대회** | ⭐⭐⭐ 권장 | ⭐⭐ 보조 |
| **타 경진대회** | ⭐⭐ 일반 | ⭐⭐⭐ 권장 |

**권장 전략**: 건강계좌를 심평원 경진대회 주력, CocFlow를 별도 의료 AI 경진대회에 병행
