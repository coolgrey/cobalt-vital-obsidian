# CocFlow AI — 문제정의

## 핵심
항암 환자는 자신의 증상을 말로 표현한다.
의료진은 그 표현을 임상 데이터로 쓸 수 없다.
이 간극이 반복되는 비효율의 원인이다.

## 구조적 문제 4가지

### ① 병원 밖 증상 판단 불가
항암치료 중 환자는 부작용을 경험하지만, 병원 외 환경에서는 증상의 심각도를 스스로 판단하기 어렵다.
→ 불필요한 응급실 방문 / 반대로 치료 시기 지연

### ② 비정형 정보의 한계
전화 기반 상담은 환자의 주관적 표현에 의존 → 비정형 정보 전달 + 기록 누락
→ 의료진 의사결정에 활용 불가능한 형태로 존재

### ③ 의료진 정보 부족
병원 밖 환자의 증상을 객관적으로 파악할 데이터가 없음
→ 환자 주관적 표현에 의존한 비효율적 진료 반복
→ 의료 자원의 비효율적 사용

### ④ 진료 연속성 단절
수도권 대형병원 중심 치료 구조 → 지방 병원 방문 시 치료 이력·상태 정보 미공유
→ 진료 연속성 저하

## 기존 서비스 한계
- 설문 기반 입력 방식 → 환자의 자연어 표현 반영 못함
- 비정형 데이터를 의료진 활용 가능한 형태로 변환 불가
- 단순 상담 챗봇 → 임상 의사결정 지원 불가

## HIRA 데이터 관점
심평원 데이터가 보여주는 동일 문제:
- 암환자 외래·응급 재방문율 높음 → 예방 가능한 비효율
- 항암제 처방 후 부작용 관련 재진료 패턴 → 사전 개입 가능 지점
