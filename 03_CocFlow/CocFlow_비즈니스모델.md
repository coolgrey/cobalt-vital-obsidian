# CocFlow AI — 비즈니스 모델

## 수익 구조

| 채널 | 모델 | 단가 | 비고 |
|------|------|------|------|
| B2H (병원) | 의료기관 구독 | 월 OO만원/기관 | 주요 수익원 |
| B2B (기업) | 기업 복지 연계 | 협의 | 항암 지원 프로그램 연계 |
| 데이터 플랫폼 | 비식별 증상 데이터 제공 | 별도 협의 | 장기 확장 |

> ⚠️ 단가 미확정 — 의료기관 파일럿 후 결정 예정

## 수익화 경로

**단기:** 의료기관 파일럿 (무료 또는 소액) → 임상 데이터 확보
**중기:** 병원 구독 계약 → 반복 수익 구조
**장기:** digital phenotype 데이터 플랫폼 → B2B 확장

## 건강계좌 BM과의 차이

| | 건강계좌 | CocFlow AI |
|--|---------|-----------|
| 주 고객 | 일반인·기업 | 의료기관·암환자 |
| 핵심 가치 | 예방·인센티브 | 임상 효율·데이터 품질 |
| 수익 구조 | B2G→B2B→B2C | B2H 중심 |
| 데이터 성격 | 건강행동 데이터 | 증상·임상 구조화 데이터 |
