# 06_RAW

이 폴더는 **원문 보관소**입니다.

## 목적
- 웹 문서, 논문, 저장소, 데이터셋, 이미지 등 **가공 전 원본**을 저장
- 해석, 요약, 비교, 시사점은 여기서 하지 않음
- LLM이 이후 위키 문서로 "컴파일"할 원재료를 모아두는 공간

## 하위 폴더 권장
- `web_clips/` : 웹 클리핑 원문
- `papers/` : 논문 원문 또는 요약 전 수집본
- `datasets/` : 데이터셋 설명/메타데이터
- `repos/` : 저장소 개요, README, 링크
- `images/` : 로컬에 저장한 이미지
- `assets/` : 첨부 파일, PDF, 기타 자산

## 규칙
- 원문을 직접 편집하지 않음
- 파일명은 가능한 한 출처와 날짜가 드러나게 작성
- 해석이 필요하면 `05_리서치` 또는 프로젝트 폴더에 별도 노트 생성
