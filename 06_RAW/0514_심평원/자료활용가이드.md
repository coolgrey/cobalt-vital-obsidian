## 📞 문제 해결 가이드

### Q: "기술혁신성 평가를 어떻게 충족해야 하나?"
**A**: 
1. `양식(5부).hwp` → 기술혁신성 기준 확인
2. `보건의료빅데이터_AI활용...v2.pdf` → 참고 사례 검토
3. `HIRA_심평원_최종사업계획서.md` → 우리 기술 설명 참고
4. `HIRA_검토보고서_문제점및보완점.md` → 기술성 체크리스트

### Q: "사업화가능성은 어떻게 보여줄까?"
**A**: 
1. `윤지현사업계획서_분석및개선보고서.md` → "시장 검증" 섹션 읽기
2. `HIRA_심평원_최종사업계획서.md` → "5. 시장 분석" + "8. 재정 계획" 확인

### Q: "다른 기관에도 제출할 건데?"
**A**: 
1. `TalkFile_제2026_208호...pdf` → 다중 기관 전략 확인
2. `05_리서치/창업지원프로그램_현황.md` → 기관별 요구사항 비교
3. `HIRA_심평원_최종사업계획서.md` → 기본 템플릿으로 사용
