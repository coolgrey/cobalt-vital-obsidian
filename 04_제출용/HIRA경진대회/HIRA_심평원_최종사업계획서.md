# 심평원 사업계획서
## 「건강계좌」: 건강검진 데이터를 자산으로 만드는 AI 건강관리 플랫폼

---

## 1. 사업 개요

### 1.1 사업명
**건강계좌 (HealthAsset)** — 공공 건강검진 데이터 + 웨어러블 + 행동 데이터를 통합하여 개인 건강자산을 추적·관리·행동변화로 연결하는 AI 기반 건강증진 플랫폼

### 1.2 사업의 핵심 가치
- **문제**: 건강검진 이상소견자 1,044만명 중 59.8%가 검진 후 일상 관리 구간이 완전히 공백
- **솔루션**: 공공 PHR(건강검진) + 웨어러블 라이프로그 + 행동 데이터를 AI로 통합하여 건강자산지수 산출
- **인센티브**: 행동변화에 직접적인 경제적 보상 (포인트 적립 → 현금화)
- **사회적 가치**: 건강보험 재정 위기의 근본 원인인 "예방의료 공백"을 데이터 기반으로 해결

### 1.3 창업자 배경
| 구성 | 인물 | 경력 | 역할 |
|------|------|------|------|
| 대표 | 최지원 (Chris) | UX 16년 (금융·의료기기·DTx) | 제품 기획 / 사업전략 |
| 공동창업자 | 윤지현 | 임상간호사 (RN/OCN), 상급종합병원 10년 | 임상 자문 / 도메인 전문성 |

**팀 강점**: UX × 임상의 조합으로 사용성과 의학적 신뢰성을 동시에 구현

## 2. 사업의 필요성 (문제정의 + 시장 기회)

### 2.1 구조적 문제: 3가지 "절벽"

#### ① 인구의 절벽
- 2025년 65세 이상 인구 비중 20% 진입 (초고령사회 돌입) → 부양 인구 감소 + 돌봄 수요 급증
- 고령자 1인당 연간 진료비: 551만원 (건강보험심사평가원, 2024)

#### ② 비용의 절벽
- 만성질환 진료비가 전체의 **80.3% 집중**
- 건강보험 누적 적자 2024년 말 기준 10조원대 도달
- 예방의료로의 패러다임 전환 필요 (치료 중심 → 예방 중심)

#### ③ 관리의 절벽 (가장 중대)
| 지표 | 수치 | 출처 |
|------|------|------|
| 건강검진 수검자 | 1,746만명 | 건보공단 2023 |
| 이상소견 발견율 | 59.8% (1,044만명) | 건보공단 2023 |
| 혈당 조절 목표 달성률 | 25% (4명 중 1명) | 질병관리청 2023 |
| 검진 후 일상 관리 연계 구조 | **0%** | 현황 분석 |

**핵심 공백**: 검진 후 365일의 일상 건강관리 구간이 완전히 방치됨

### 2.2 데이터 파편화 현황

현재 국내 건강 데이터는 세 개의 고립된 사일로에 존재:

| 데이터 | 보유 기관 | 현황 | 문제 |
|--------|--------|------|------|
| **건강검진 (PHR)** | 건보공단 | 연 1회 스냅샷 | 검진 후 일상 연결 안 됨 |
| **웨어러블** | 개인 (Apple, Garmin 등) | 실시간 라이프로그 | 검진 수치와 통합되지 않음 |
| **금융/보험** | 보험사 | 가입 정보만 | 건강 행동과 보상 연결 모델 없음 |

**결과**: 데이터는 풍부하지만 **통합·해석·행동변화로의 연결이 불가능**

### 2.3 기존 경쟁사의 공백 분석

| 서비스 | 타겟 | 장점 | 공백 |
|--------|------|------|------|
| 나만의닥터 | 비대면 진료 | 의료진 상담 | 예방 단계 (검진 이후) |
| 카카오헬스케어 | 질환관리 | 이미 확진자 관리 | 검진 후 무증상 구간 |
| 룰루메딕 | 복약 관리 | 약물 복용 추적 | 약물 치료 중인 환자만 |

**결론**: **검진 이후 예방관리 구간 (=가장 개입 효과가 큰 시점)은 완전한 공백**

### 2.4 심평원 정책 기회
- 건강보험 재정 위기 → 2025년 예방의료 정책 급전환
- 건강정보 고속도로 본격 가동 (2023~, 1,263개 의료기관 연계)
- 디지털의료제품법 시행 (2025.1.24) → 비의료 건강관리 서비스 즉시 출시 가능
- 보험사들의 건강증진형 상품 개발 러시 → B2B 파트너십 수요

## 3. 사업 내용 및 기술

### 3.1 핵심 솔루션: 건강계좌 (HealthAsset)

금융 계좌의 구조를 건강에 적용한 개념:
- **토스/뱅크샐러드**가 금융결제원의 공공 레일을 활용한 것처럼
- **건강계좌는 건강정보 고속도로 위에서 건강자산 관리 UX를 구현**

#### 사용 흐름
```
1단계: 데이터 수집
  → 건강검진 (공공 PHR) + 웨어러블 (걸음수, 수면, 심박) 통합 수신

2단계: AI 처리
  → 3-레이어 가중치 알고리즘으로 "건강자산지수" 산출

3단계: 시각화 + 행동변화
  → 등급(Platinum/Gold/Silver/Care)으로 표시
  → 건강 미션 제시 (운동, 식이, 수면 등)

4단계: 인센티브
  → 미션 달성 → 포인트 적립 → 현금화 / 보험료 할인
```

### 3.2 핵심 기술: AI 건강자산지수

#### 알고리즘 구조 (3-레이어 가중치)

```
건강자산지수 = PHR(50%) × 웨어러블(35%) × 행동이행도(15%)

┌─────────────────────────────────────────┐
│ 레이어 1: PHR (건강검진) — 50%           │
├─────────────────────────────────────────┤
│ • 혈압 (수축/이완 혈압)                  │
│ • 혈당 (공복혈당, HbA1c)                │
│ • 지질 (콜레스테롤, 중성지방)            │
│ • 폐기능 (FEV1 %)                       │
│ (각 지표별 정상/경고/위험 3단계 평점)   │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 레이어 2: 웨어러블 라이프로그 — 35%      │
├─────────────────────────────────────────┤
│ • 일일 걸음수 (목표: 8,000~10,000)       │
│ • 수면 시간 (목표: 6.5~8시간)            │
│ • 심박 변이도 (스트레스 지표)            │
│ (30일 이동평균 추적)                     │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 레이어 3: 행동 이행도 — 15%              │
├─────────────────────────────────────────┤
│ • 주간 건강 미션 달성률 (목표: 70%)      │
│ • 앱 활성 일수 (월간 로그인 일수)        │
│ • 건강 기록 입력 일관성                  │
└─────────────────────────────────────────┘

→ 최종 출력: Platinum(90~100) / Gold(75~89) / Silver(60~74) / Care(~59)
```

#### 의학적 근거
- **서울대병원 + 네이버 공동 연구** (JMIR 저널, 2025)
- 151,281명의 건강검진 데이터 기반 검증
- 각 지표의 가중치 및 임계값이 대규모 임상데이터로 입증됨

#### 글로벌 레퍼런스
| 사례 | 국가 | 성과 | 근거 |
|------|------|------|------|
| National Steps Challenge | 싱가포르 | 170만명 참여, ROI 145% | AHA Journal |
| b.well | 미국 | 의료+웨어러블+금융 통합 | OpenAI·Google 파트너 |

### 3.3 데이터 수집 로드맵 (3단계)

#### 0단계: MVP (2026.04~)
**App-to-App FHIR 기반 수집**
- 사용자가 앱 내 버튼으로 건강검진 10년치 조회 (서울대병원 등과 연동)
- 웨어러블(Apple Health, Google Fit) 연동
- FHIR R4 파서 구현 완료 (설계 단계)

#### 1단계: 정식화 (2027 H1)
**건강정보 고속도로 정식 등록**
- 건보공단 공식 PHR 플랫폼 등록
- 진료이력·투약이력 자동 수신 (사용자 동의 기반)
- 1,263개 의료기관의 진료 정보 실시간 수집 가능

#### 2단계: 통합 인프라 (2028~)
**건강 + 웨어러블 + 금융 B2B 통합 인프라**
- 기업복지 플랫폼과 연동
- 보험사 건강증진형 상품 연계
- 금융사 개인신용평가 데이터 활용

### 3.4 기술적 차별성 (해자)

| 구분 | 우리 접근 | 차별성 |
|------|---------|-------|
| **데이터 보안** | 원본 데이터 개인 기기 보관, 서버엔 결과값만 저장 | 개인정보 침해 최소화 |
| **규제 적합성** | 모든 경로가 법적 기반 준비 (FHIR → 고속도로 → DTx법) | 즉시 합법적 출시 가능 |
| **의료진 신뢰** | 서울대병원 임상 검증 데이터 기반 | 엔터프라이즈급 신뢰성 |
| **API 우위** | 공공 PHR 레일 (토스처럼) | 독점이 아닌 공공재 기반 확장성 |

## 4. 비즈니스 모델 및 기대효과

### 4.1 3축 수익 구조 (B2G → B2B → B2C)

#### 1단계: B2G (공공기관) — 초기 현금흐름 확보
| 고객 | 모델 | 단가 | 1년차 목표 | 3년차 목표 |
|------|------|------|-----------|-----------|
| 지자체·보건소·시범도시 | 건강관리 시범사업 수주 | 건당 2,000~3,500만원 | 2,000만원 | 2억원 |

**사례**: 
- 성남시/판교 지역 건강증진 시범 사업 (성남창업센터 네트워크 활용)
- 시·도 보건소 만성질환 예방관리 서비스

#### 2단계: B2B (기업복지) — 스케일 확장
| 고객 | 모델 | 단가 | 1년차 목표 | 3년차 목표 |
|------|------|------|-----------|-----------|
| 중견기업·대기업 HR팀 | 임직원 건강관리 앱 (구독형) | 월 3,000~5,000원/인 | 1,000만원 | 2억원 |

**접근**: 기존 건강검진 기업(삼성, LG, SK 등)과 자사 검진 데이터 연동

#### 3단계: B2C (개인) — 최종 목표
| 고객 | 모델 | 단가 | 목표 |
|------|------|------|------|
| 개인 (MZ세대·건강 관심층) | 프리미염 구독 (월) | 월 9,900원~ | 2억원 (2027~) |

**자기강화 구조**: B2G 성공 → B2B 확장 → B2C 신뢰 구축

### 4.2 인센티브 재원 구조

#### MVP (자체 포인트)
- 앱 내 운영 포인트로 운영
- 추후 현금화 경로 추가 (카드사 제휴 등)

#### B2B (기업 복지비)
- 기업이 임직원 건강관리 투자로 부담
- 복지 비용의 일부로 처리

#### B2G + 보험사 연계 (최종 단계)
- 보험사의 손해율 감소분 일부를 사용자에게 환원
- 예: 혈압 관리 성공 → 보험료 할인 5% (보험사 수익성 유지하며 행동변화 유인)

### 4.3 시장 규모 및 재무 목표

#### 시장 분석 (기준: 2023년 기준)
```
TAM (전체 시장)
= 국내 디지털헬스케어 시장 (2024년) 약 4.7조원

SAM (유효 시장)
= 건강검진 이상소견자 (1,044만명) × 스마트폰 사용 비율 62% × 연간 ARPU 118,800원
= 약 7,700억원

SOM (우리 목표, 2027년)
= 수도권 집중 + 초기 수용층 5% 점유율
= 약 11억원 ~ 20억원 (B2B 기업복지 확대 시)
```

#### 재무 목표
| 연도 | 매출 | 구성 | 상태 |
|------|------|------|------|
| 2026 | 2,000만원 | B2G 시범 1건 + MVP 운영 | MVP 런칭 |
| 2027 | 11~20억원 | B2G 1~2건 + B2B 3~5개사 | 정식 출시 |
| 2028 | 50~100억원 | B2G 3~5건 + B2B 10+개사 + B2C 초기 | 시리즈A 펀딩 |

#### 손익분기점
- 월 사용자 5,000명 도달 시 손익분기점 예상 (월 MAU 기준)
- 목표: 2027년 내 달성

### 4.4 사회적 기대효과

#### 공중보건 관점
| 효과 | 수치화 |
|------|--------|
| 건강보험 진료비 절감 | 고위험군 1인당 연 500만원 관리시 20~30% 절감 |
| 만성질환 예방 | 고혈압/당뇨 조절률 25% → 40% 상향 시 의료 부담 2조원대 감소 |

#### 심평원의 관심 지점

- 보험사 + 기업복지 + 공공보건의 3자 통합 모델로 정책 실현성 입증

## 5. 추진 계획 (로드맵)

### 5.1 2026년 5대 추진 단계

| 단계 | 기간 | 핵심 과제 | 달성 기준 |
|------|------|---------|---------|
| **1단계** | ~2026.05 | 법인설립 + 개발 파트너 확정 | 법인 등록 완료 (AC 배치 투자금 수령 조건) |
| **2단계** | 2026.06~08 | MVP 개발 진행 (FHIR 파서, 앱 UI/UX) | 원본 설계 완료 (23개 화면) |
| **3단계** | 2026.09~10 | 베타 런칭 + 피드백 수집 (사용자 100~200명) | 앱 스토어 배포, 초기 CMO 달성 |

## 9. 결론

### 9.1 사업의 본질
건강 데이터는 이미 존재한다.  
**부재한 것은 그것을 해석·추적·행동변화로 연결하는 운영체계다.**

건강계좌는:
- ✅ **공공 인프라 기반** (건보공단 PHR, 고속도로)
- ✅ **과학적 근거** (서울대 임상 검증, JMIR 논문)
- ✅ **규제 적합** (DTx법 2025년 시행)
- ✅ **팀 역량** (UX 16년 + 임상 10년)
- ✅ **현장 필요** (심평원의 핵심 정책 방향)

의 5가지를 갖춘 타이밍 비즈니스다.

### 9.2 심평원을 통한 기대효과
이 사업이 성공하면:
1. **공공 PHR(고속도로)의 실질적 활용** 사례 창출
2. **건강보험 재정 위기의 데이터 기반 해결** 모델 제시
3. **예방의료 → 행동변화 → 재정 절감**의 선순환 구조 증명
4. **정부 정책(DTx법, PHR 전개)**의 실제 임팩트 검증

### 9.3 향후 비전 (2028년)
```
건강계좌 (검진 후 예방)
         +
CocFlow AI (진료 중 증상 관리)
         =
코발트바이탈 플랫폼
```
