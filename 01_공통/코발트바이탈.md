# 코발트바이탈 (CobaltVital)

## 회사 개요
- 설립 예정: 2026년 5월
- 대표: 최지원
- 소재지: 경기도 성남시 분당구
- 사무공간: 판교 창업존 서류 탈락 (2026.03) → 대안 공간 검토 중
- 사업 분야: 디지털 헬스케어 / 예방의료 / 헬스-금융 융합

## 미션
"병원 밖 99%의 일상에서 건강 데이터를 자산으로 만든다"

## 두 서비스의 관계

```
코발트바이탈
├── 건강계좌 (B2C·B2B)
│   대상: 일반인 / 검진 후 방치 구간
│   포지션: 예방 — 검진 이후 일상 전구간
│
└── CocFlow AI (B2B·B2H)
    대상: 암환자 + 의료진
    포지션: 치료 과정 — 병원 밖 증상 관리
```

**왜 두 서비스가 함께인가:**
최지원(UX 16년)의 "사람이 실제로 쓰는 서비스 설계"와
윤지현(RN·OCN)의 "임상 현장에서 검증된 데이터 구조"가
예방(건강계좌)과 치료과정(CocFlow)에 각각 적용된다.

한 회사에서 두 서비스가 나온 이유는 두 창업자가 같은 구조적 문제를
각자의 현장에서 반복 목격했기 때문이다.
