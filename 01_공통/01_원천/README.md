## 핵심 분석 문서

### 전략 분석 (Strategy)
- **[[01_공통/01_원천/문제정의|문제정의]]** — 시장의 중심 문제 정의
- **[[01_공통/01_원천/사용자|사용자]]** — 타겟 사용자 페르소나 및 니즈
- **[[01_공통/01_원천/포지셔닝|포지셔닝]]** — 시장 포지셔닝 전략

### 시장 분석 (Market)
- **[[01_공통/01_원천/시장분석|시장분석]]** — 시장 규모, 성장성, 기회
- **[[01_공통/01_원천/경쟁사분석|경쟁사분석]]** — 경쟁사 기술 및 시장 동향
- **[[01_공통/01_원천/솔루션|솔루션]]** — 기존 솔루션 벤치마크

## 활용 원칙

### 문서의 역할
- **원본 소스**: 모든 프로젝트의 비즈니스 논리를 지원하는 기초 자료
- **공유 자산**: 프로젝트 간 중복을 제거하고 일관성 유지
- **지속 업데이트**: 시장 동향 변화에 따라 정기적으로 갱신

### 참고 방식
각 프로젝트는 자신의 폴더(02_건강계좌, 03_CocFlow)에서 01_원천 문서를 참고하여 프로젝트 특화 분석 작성

### 예시 링크 구조
```
01_공통/01_원천/시장분석 (공통 시장 분석)
  ↓ 참고
02_건강계좌/건강계좌_시장기회 (건강계좌 특화 시장 분석)
  ↓ 참고
04_제출용/성남창업센터 (제출용 최종 자료)
```
