# 비즈니스모델

## 핵심 구조 한 줄

> 사용자의 건강행동 데이터가 쌓일수록,
> B2B(기업·보험사)에서 받는 단가가 올라가는 구조다.

사용자에게 인센티브를 주는 비용을, B2B 파트너가 부담한다.
이게 건강계좌 BM의 핵심이다.

---

## 수익 구조 — 3단계 순서

### 1단계: B2C 구독 (MVP ~ 2026년 말)

**목적:** 초기 사용자 확보 + 행동 데이터 축적

| 구분 | 내용 |
|------|------|
| 무료 티어 | 검진 결과 입력 + 기본 행동 추천 |
| 유료 티어 | 월 9,900원 — 개인화 가이드 + 포인트 2배 + 상세 리포트 |
| 인센티브 재원 | 초기엔 자체 포인트 (현금화 불가, 앱 내 사용) |

> 가격 근거: 설문 Q9 결과로 최종 결정. 현재 가설은 7,000~12,000원 구간.
> 참고: 룰루메딕 혈당멤버십, 닥터다이어리 구독 모델 모두 이 구간 운영 중.

---

### 2단계: B2B 기업복지 패키지 (2027년~)

**목적:** 매출 규모화 + 단가 상승

닥터다이어리가 2025년 대기업·공공기관·금융권 60여개 기관과 협력하며 B2B 헬스케어 500% 성장을 기록한 것처럼, 임직원 건강관리는 B2B 수요가 검증된 시장이다.

**건강계좌의 B2B 제안:**

| 구분 | 내용 |
|------|------|
| 상품명 | 건강계좌 기업 웰니스 패키지 |
| 대상 | 직원 100명 이상 기업 HR/복지 담당자 |
| 과금 방식 | 1인당 월 3,000~5,000원 (연간 계약) |
| 기업이 받는 것 | 임직원 건강관리 대시보드 + 건강개선도 리포트 |
| 임직원이 받는 것 | 유료 기능 무료 + 포인트 추가 적립 |

**B2B 첫 타겟:**
- 성남/판교 IT 기업 (접근성 우선)
- 중소기업 50~200인 (의사결정 빠름)
- 공공기관/보건소 (분당보건소 협력 레퍼런스 활용)

---

### 3단계: 보험사 파트너십 (2027년~)

**목적:** 단가 최대화 + 해자 구축

건강증진형 보험상품 가이드라인은 2017년부터 존재하고, 생손보 16개 이상 회사가 이미 출시했다. 보험사 입장에서 건강행동 데이터는 손해율 감소로 직결된다.

**건강계좌의 보험사 제안:**

| 구분 | 내용 |
|------|------|
| 우리가 제공하는 것 | 검증된 건강행동 데이터 (사용자 동의 기반) |
| 보험사가 제공하는 것 | 보험료 할인 + 사용자 인센티브 재원 |
| 과금 방식 | 데이터 API 라이선스 + 성과 기반 수수료 |
| 사용자 혜택 | 보험료 실질 할인 (→ 금전적 보상의 최고 형태) |

> 이 단계에서 건강계좌의 인센티브가 "앱 포인트"에서 "실제 보험료 할인"으로 격상된다.
> 이것이 경쟁사가 복제하기 어려운 해자다.

---

## 인센티브 재원 구조 (핵심 설계)

사용자에게 돌려주는 인센티브는 어디서 오는가.

```
단계 1 (MVP)
사용자 유료 구독료 일부 → 포인트 재원
(앱 내 사용 포인트, 현금화 없음)

단계 2 (B2B)
기업 구독료 일부 → 임직원 포인트 재원
(기업이 임직원 건강관리 비용 부담)

단계 3 (보험사)
보험사 데이터 라이선스 수수료 → 보험료 할인 재원
(보험사 손해율 감소 이익의 일부를 사용자에게 환원)
```

## 단위 경제 (Unit Economics) — 가설

> 아래 수치는 전부 가설이다. 베타 이후 실측값으로 교체 필요.

### B2C

| 지표 | 가설값 | 근거 |
|------|--------|------|
| ARPU (월) | 6,000원 | 유료 전환율 30% × 9,900원 + 무료 70% |
| CAC (사용자 획득 비용) | 5,000~15,000원 | 헬스케어 앱 업계 평균 추정 |
| LTV (12개월 기준) | 72,000원 | ARPU 6,000원 × 12개월 |
| LTV/CAC | 5~14배 | 목표: 3배 이상 유지 |
| 4주 리텐션 목표 | 40% | MVP 성공 기준 |

### B2B

| 지표 | 가설값 |
|------|--------|
| 계약 단가 | 1인당 월 3,000~5,000원 |
| 최소 계약 인원 | 50인 |
| 계약당 월 매출 | 150,000원~250,000원 |
| 연간 계약 매출 | 1,800,000원~3,000,000원 |
| 목표 계약사 수 (2027년 말) | 10개사 |

## 비용 구조

| 항목 | 비중 | 비고 |
|------|------|------|
| 개발/운영 | ~40% | 초기 외주, 이후 내재화 |
| 마케팅/사용자 획득 | ~30% | 초기 집중 투자 후 유기적 성장 |
| 인센티브 재원 | ~15% | 단계별 외부 조달로 전환 |
| 인건비 | ~10% | 창업자 2인 + 채용 계획 |
| 기타 (법무·서버 등) | ~5% | |

## BM 플라이휠

```
사용자 증가
    ↓
건강행동 데이터 축적
    ↓
B2B 제안 설득력 상승
    ↓
B2B 계약 → 인센티브 재원 확보
    ↓
사용자 인센티브 강화
    ↓
사용자 증가 (반복)
```

이 루프가 돌기 시작하면 경쟁사가 데이터 없이 진입하기 어려워진다.
